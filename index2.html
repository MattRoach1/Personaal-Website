Description of NBA 2k15

NBA 2k15 is basically where the fan base of NBA 2k games started. The game mode called, myPARK was first introduced in this game. This certain mode allows you to use your created player to compete against other online players by playing in pickup games of 3v3,2v2,and 21. If this game did not come out with myPARK then NBA 2k wouldn't be how it is now. The current Price of this game is $59.99.
